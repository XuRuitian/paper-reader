# paper-reader

精读学术文献的专家级 Skill，支持 PDF 全文提取、六维度深度分析和 Word 报告生成。

## 功能

- **PDF 全文提取**：自动提取论文文字、图表、表格内容
- **研究灵感时刻**：从论文中提炼启发性灵感，激发新的研究思路
- **六维度深度分析**：研究目标、新方法、实验验证、未来方向、批判分析、实用建议
- **Word 报告生成**：一键输出结构化精读报告

## 文件结构

```
paper-reader/
├── SKILL.md                        # Skill 说明文档
├── scripts/
│   ├── extract_text.py             # PDF 全文提取脚本
│   └── generate_report.js          # Word 报告生成脚本
└── references/
    └── academic_prompt.md           # 六维度分析框架
```

## 安装

```bash
clawhub install paper-reader-xuruitian
```

## 使用

在支持 Skill 的 AI 助手中，加载 paper-reader Skill 后，上传 PDF 即可开始精读。
